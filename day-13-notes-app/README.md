# Notes App (React + Flask + SQLite)

A simple **Notes App** with full CRUD functionality using:
- **Backend:** Flask + SQLite
- **Frontend:** React
- **Database:** SQLite (auto-created)

---

## Features
- Add new note
- View all notes
- Edit note
- Delete note

---
